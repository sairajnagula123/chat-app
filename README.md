# 💬 Real-Time Chat Application

A full-stack real-time chat application built using the **MERN Stack** and **Socket.IO**. The application enables users to communicate instantly with features like authentication, online status, typing indicators, and message seen status.

## 🚀 Features

* User Registration & Login
* JWT Authentication
* Protected Routes
* Real-Time Messaging with Socket.IO
* Online/Offline User Status
* Typing Indicator
* Message Seen Status
* Conversation Management
* Responsive UI

## 🛠️ Tech Stack

### Frontend

* React
* React Router
* Axios
* Socket.IO Client
* CSS

### Backend

* Node.js
* Express.js
* Socket.IO
* JWT
* bcrypt

### Database

* MongoDB
* Mongoose

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/chat-app.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```
